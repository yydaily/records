C语言入门(基础知识)

2021-05-31

C++语言笔记

基础知识

变量及赋值

  1. 变量就是可以变化的量，每个变量都会有一个名字。使用变量之前必须先定义变量。

  2. 变量的基础数据类型一般来讲有三种：整型，浮点型，字符型。

  3. 整型数据是指不带小数的整数，常用包括（int， short，unsigned int，unsigned long，long long等）。典型的是 int 和 long long，前者的范围是[−231,231−1][-2^{31}, 2^{31}-1][−231,231−1]，大概范围是[-2\*10^9, 2\*10^9]，后者的范围是 [−263,263−1][-2^{63}, 2^{63}-1][−263,263−1]，大概范围是 [-9\*10^{18}, 9\*10^{18}]。

  4. 浮点数是指带小数的实数，常用的包括（float，double）。double的范围大概是 [-1.7\*10^{308},1.7\*10^{308}]

  5. 字符型是指可以由各种字符来组成的字符串，可以简单理解为一句话，这句话可以是中文，可以是英文，可以是各种可以表示的语言类型。

常量

  1. 在程序执行过程中，值不发生改变的量，成为常量

  2. 常量通常直接用来计算/输出，例如 cout << 1.2<<endl;

格式化输入输出

  1. 格式化输出语句，是指将各种类型的数据按照格式化后的类型及制定位置在计算机上显示出来。

算数运算符

名称 运算符号 举例
加法运算符 + 2+10 = 12
减法运算符 - 2 - 1 = 1
乘法运算符 * 2 * 3 = 6
除法运算符 / 10 / 3 = 3
求余运算符 % 23 % 3 = 2
自增运算符 ++ int a = 1; a++
自减运算符 – int a = 2; a–

  1. 如果相除的两个数都是整数的话，结果也会是整数，忽略小数部分。

  2. 取余只能用于两个整数之间，浮点数和整数没法进行取余操作。

  3. 没有乘方、开方等操作。

关系运算符

符号 意义 举例 结果
> 大于 1>2 0
>= 大于等于 1 >=1 1
< 小于 2 < 1 0
<= 小于等于 1 <= 1 1
== 判断相等 1 == 2 0
!= 判断不相等 1 != 2 1
  1. $>=,\ <=,\ ==,\ !=\ $等符号之间不能有空格。

逻辑运算符

符号 意义 说明 举例 结果
&& 逻辑与 同真为真，否则为假 0 && 1 0
|| 逻辑或 有真为真，否则为假 0 || 1 1
! 逻辑非 真假取反 !0 1

  1. 逻辑与，只有参与运算的两个条件都为真时，结果才为真，否则为假。例如 5>=5 && 2>1，结果为真。 4>=1 && 3<2，结果为假。2 > 3 && 3 < 1，结果为假。

  2. 逻辑或，只要参与运算的两个条件有一个为真时，结果就为真，否则为假。例如 1>=1 || 2 > 3，结果为真。1>3 || 3>5，结果为假。

  3. 逻辑非，对参与运算的结果取反。例如 !(2>4)，结果为真。
