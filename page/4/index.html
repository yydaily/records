深度优先遍历

定义

按照惯例，参考wiki：

Depth-first search (DFS) is an algorithm for traversing or searching tree or graph data structures. The algorithm starts at the root node (selecting some arbitrary node as the root node in the case of a graph) and explores as far as possible along each branch before backtracking.

翻译过来就是：

DFS 是一种在图/树上遍历或搜索的算法。这个算法从一个根（在图中可以选择一个点作为根）出发，尽可能搜索到最远处，然后退回，继续下一个分支。

简单点说，就是解集的空间是有限的，我们按照一个方向不断的枚举每一个可能的解，直到找到一个可行解为止。比如，小时候我们玩的迷宫，只要迷宫不变化，我们走遍每一条路，总是能找到出口的，深度优先搜索就是按照一个方向一直莽，如果错了就回头，直到莽出一条路来。

算法过程

算法描述

  • 根节点入栈
  • 从栈中取出第一个节点，标记为visit，并检验它是否是目标
    • 如果找到目标，则搜索结束，并回传结果
    • 枚举每个没有visit的子节点，将他们入队，重复步骤2
  • 重复步骤 222

定义

首先参考维基百科的定义：

Fermat’s little theorem states that if ppp is a prime number, then for any integer aaa, the number apa^pap − aaa is an integer multiple of ppp. In the notation of modular arithmetic, this is expressed as

ap≡a( mod p)a^p\equiv a (\bmod p) ap≡a(modp)

翻译过来就是

ap≡a( mod p)a^p \equiv a (\bmod p)ap≡a(modp)

证明

方法1

考虑二项式 Cpn=p!n!(p−n)!C_p^n = \frac{p!}{n!(p-n)!}Cpn​=n!(p−n)!p!​ ，当 n\ne p \and n\ne 0 ，可以发现分子中含有 ppp ，而分母中不含。所以 CnpC_n^pCnp​ 一定可以整除 ppp 。

然后考虑 (b+1)p(b+1)^p(b+1)p 的二项式展开，我们有

\begin{align} (b+1)^p &\equiv \sum_{i=0}^p C_p^i b^i (\bmod p)\\ &\equiv C_p^0b^0 + C_p^pb^p(\bmod p)\\ &\equiv 1+b^p(\bmod p) \end{align}

同理，我们将 a=b−1a=b-1a=b−1 代入并一直迭代下去可以得到

\begin{align} a^p &\equiv 1+(a-1)^p (\bmod p) \\ &\equiv 1+ 1 + (b-2)^p (\bmod p)\\ &\vdots\\ &\equiv \underbrace{1+1+\dots+1}_{a}(\bmod p)\\ &\equiv a(\bmod p)\\ \end{align}

\begin{align} (a_0+1)^p &= \sum_{i=0}^p C_p^ia_0^i\\ &= a_0^p + \sum_{i=0}^{p-1} C_p^ia_0^i\\ &\equiv a_0 + \sum_{i=0}^{p-1} C_p^ia_0^i (\bmod p)\\ &\equiv a_0 + C_p^0a_0^0 (\bmod p)\\ &\equiv a_0+1(\bmod p) \end{align}

\begin{align} (a*b)-(a*c) &= a*(b-c)\\ &\equiv a*(b-c)(\bmod p)\\ &\not\equiv a*0 (\bmod p)\\ &\equiv 0 (\bmod p) \end{align}

定义

逆元也称为 模逆元 ，一个整数 aaa 和同余 nnn 的逆元是指满足一下公式的整数 bbb :

a−1≡b   ( mod n)a^{-1} \equiv b \ \ \ (\bmod n) a−1≡b   (modn)

也可以写成：

ab≡1   ( mod n)ab \equiv 1\ \ \ (\bmod n) ab≡1   (modn)

逆元存在的充要条件是 aaa 和 nnn 互素。

证明

充要性

也即证明 ∃b,ab≡1   ( mod n)⇒gcd(a,n)=1\exists b, ab\equiv 1\ \ \ (\bmod n) \Rightarrow gcd(a,n)=1∃b,ab≡1   (modn)⇒gcd(a,n)=1

\begin{align} &\exists b, ab\equiv 1 (\bmod n)\\ \Rightarrow& \exists x,\exists y, ax + ny \equiv 1 (\bmod n)\\ \Rightarrow& gcd(a, n) \le 1\\ \Rightarrow& gcd(a, n) = 1 \end{align}

其中有一步的原因是，我们在扩展欧几里得里面提到过 ax+nyax+nyax+ny 能组成的最小正整数为 gcd(a,n)gcd(a, n)gcd(a,n) ，所以如果能组成 111 ，那么说明 gcd(a,n)≤1gcd(a, n) \le 1gcd(a,n)≤1 。

也即证明 gcd(a,n)=1⇒∃b,ab≡1( mod n)gcd(a, n) = 1 \Rightarrow \exists b, ab \equiv 1 (\bmod n)gcd(a,n)=1⇒∃b,ab≡1(modn) 。

根据扩展欧几里得，一定存在 (x0,y0)(x_0, y_0)(x0​,y0​) 使得 ax0+ny0≡1( mod n)ax_0 + ny_0 \equiv 1 (\bmod n)ax0​+ny0​≡1(modn) 。这里我们令 b=x0b=x_0b=x0​ 即可。

和欧几里得类似，因为有 gcd(a,n)=1gcd(a,n)=1gcd(a,n)=1 ，所以我们可以直接套用欧几里得定理：

更进一步的，比赛的时候，经常会给出 nnn 是个质数，对于这种情况，我们通常会用费马小定理+快速幂，快速的算出一个数的逆元。也即

1. 花费理论

如下图所示，从一个城市走到另一个城市有一个花费，矩阵乘法实际就指花费多少。

C D
A 1 2
B 2 5

分别表示：

  1. 从A城市走到C城市，花费为 111
  2. 从A城市走到D城市，花费为 222
  3. 从B城市走到C城市，花费为 222
  4. 从B城市走到D城市，花费为 555

同理，第二个矩阵有

E F G
C 3 5 2
D 4 1 4

这里不再赘述，然后结果是什么呢？显然就是从一个城市走到一个城市，再走到下一个城市的花费。

例如从 A 到 E，一共有两条路径：

  1. 从A到C（花费为1），从C到E（花费为3）。
  2. 从A到D（花费为2），从D到E（花费为4）。

所以自然的花费也就有两个子项组成，也即总花费为 1×3+2×4=111\times 3 + 2 \times 4 = 111×3+2×4=11

2. 列理论

我们将矩阵拆分，可以发现一个非常有趣的现象。

我们将后一个矩阵拆成多个矩阵，将前一个矩阵当做变换矩阵，可以得到

[1225][34]=[1126][1225][51]=[715][1225][24]=[1024]\begin{bmatrix} 1&2\\2&5 \end{bmatrix} \begin{bmatrix} 3\\4 \end{bmatrix}= \begin{bmatrix} 11\\26 \end{bmatrix}\\\\ \begin{bmatrix} 1&2\\2&5 \end{bmatrix} \begin{bmatrix} 5\\1 \end{bmatrix}= \begin{bmatrix} 7\\15 \end{bmatrix}\\\\ \begin{bmatrix} 1&2\\2&5 \end{bmatrix} \begin{bmatrix} 2\\4 \end{bmatrix}= \begin{bmatrix} 10\\24 \end{bmatrix}\\ [12​25​][34​]=[1126​][12​25​][51​]=[715​][12​25​][24​]=[1024​]

然后就能发现，这不就是个平面空间的坐标变化吗？左边矩阵定义了变换规则，右边矩阵是起点，等号右边是重点，所以矩阵乘法是什么？

简单点说就是：坐标变换 。

这也迎合了之前说的，矩阵的运算就是运动。
