Shuffle’m Up

开启传送门

题目描述

A common pastime for poker players at a poker table is to shuffle stacks of chips. Shuffling chips is performed by starting with two stacks of poker chips, S1S_1S1​ and S2S_2S2​, each stack containing CCC chips. Each stack may contain chips of several different colors.

The actual shuffle operation is performed by interleaving a chip from S1S_1S1​ with a chip from S2S_2S2​ as shown below for C=5C = 5C=5:

The single resultant stack, S12S_{12}S12​, contains 2∗C2 * C2∗C chips. The bottommost chip of S12S_{12}S12​ is the bottommost chip from S2S_2S2​. On top of that chip, is the bottommost chip from S1S_1S1​. The interleaving process continues taking the 222nd chip from the bottom of S2S_2S2​ and placing that on S12S_{12}S12​, followed by the 222nd chip from the bottom of S1S_1S1​ and so on until the topmost chip from S1S_1S1​ is placed on top of S12S_{12}S12​.

After the shuffle operation, S12S_{12}S12​ is split into 222 new stacks by taking the bottommost CCC chips from S12S_{12}S12​ to form a new S1S_1S1​ and the topmost CCC chips from S12S_{12}S12​ to form a new S2S_2S2​. The shuffle operation may then be repeated to form a new S12S_{12}S12​.

For this problem, you will write a program to determine if a particular resultant stack S12S_{12}S12​ can be formed by shuffling two stacks some number of times.

Input

The first line of input contains a single integer N,(1≤N≤1000)N, (1 \le N \le 1000)N,(1≤N≤1000) which is the number of datasets that follow.

Each dataset consists of four lines of input. The first line of a dataset specifies an integer C,(1≤C≤100)C, (1 \le C \le 100)C,(1≤C≤100) which is the number of chips in each initial stack (S1S_1S1​ and S2S_2S2​). The second line of each dataset specifies the colors of each of the CCC chips in stack S1S_1S1​, starting with the bottommost chip. The third line of each dataset specifies the colors of each of the CCC chips in stack S2S_2S2​ starting with the bottommost chip. Colors are expressed as a single uppercase letter (AAA through HHH). There are no blanks or separators between the chip colors. The fourth line of each dataset contains 2∗C2 * C2∗C uppercase letters (AAA through HHH), representing the colors of the desired result of the shuffling of S1S_1S1​ and S2S_2S2​ zero or more times. The bottommost chip’s color is specified first.

Output

Output for each dataset consists of a single line that displays the dataset number (111 though NNN), a space, and an integer value which is the minimum number of shuffle operations required to get the desired resultant stack. If the desired result can not be reached using the input for the dataset, display the value negative 1(−1)1 (−1)1(−1) for the number of shuffle operations.

Sample Input

2

4

AHAH

HAHA

HHAAAAHH

3

CDE

CDE

EEDDCC

Sample Output

1 2

2 -1

题意

题意远比题目难，当然也是我比较懒的缘故，说实话题目是长了点，其实意思还是很好懂的：就是说给你两个字符串，然后你要进行操作，操作过程就是将两个字符串洗一下，洗成一个字符串（就像洗扑克牌一样）。具体洗的过程是这样，将第二个字符串的第一个字符放在新生成的字符串的第一个，第一个字符串的第一个字符放在新生成的字符串的第二个，以此类推。
比如：
s1：12345
s2：67890
那么新生成的字符串是s12:6172839405

然后把新生成的字符串从中间拆成两段，分别是s1，s2，然后反复的这样进行下去，直到和给定的第三个字符串相等未知，输出最小的操作次数，如果无论交换多少次都不能达成目的就输出-1。然后输出的第一个数字表示数据的组数，第二个数据表示当前组数的最少交换次数
