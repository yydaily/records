20220308

2022-03-08

不知不觉已经三个月没有写日记了，这段时间说不上忙，主要是一直懒着，忘了把博客恢复回来，所以就一拖再拖直到今天啦~

逐渐远离字节1095的生活后，开始渐渐的开心起来了。每天早睡早起的，开始逐渐的锻炼身体了。晚上偶尔熬到一点的时候就感觉身体不行了，得睡觉，也不知道是好是坏。

以前放下过好多次的刷题也逐渐开始刷起来了。对于刷题这件事，真的是经历过【好爽】->【再也不想刷题了】->【无悲无喜】的状态，现在就是一个纯放松的状态，感觉沉浸到题目里面的时候，会有一种压力的释放的感觉。

时间多了之后，现在想做的事反而更多了起来，想学画画，想学吉他，想练字，也想练打字。月初定的今年15本书的目标目前已经过去两个月了，也只看了一本书，也没写读后感，也不知道是怠惰了，还是怎么得。

20211207

2021-12-07

昨天提了离职了，属实比我预期还要早一点。来字节之前，我以为我会留下2-3年时间，没想到刚好一年半就离开了。

离开的原因和离开的理由，已经和太多人说了太多次了，已经不想在这里赘述了。只能说是每个人有自己的诉求吧，无关他人评判的优劣，始终觉得只要自己不后悔，心向往之便是极好。

剩下便来说说总结吧，权当是对过往青春的告别。

锅

翻了一下自己今年的所有日记，一共10次吐槽，基本都是自己工作搞出了锅，然后企图能记住这些错误，让自己好受一点。

  1. 大部分错误都是不够细心，或者说不够追求极致，其实很多时候是能做的更好的，但是没有做到。应该还有很多提高的地方。
  2. 做的项目都黄的差不多了，虽然决策不是我下的，但是多少还是会有点难过的。

美好瞬间

  1. 成功的把工卡弄丢了，然后换了个美美的照片换了新工卡，然后下一周找到了老工卡（但是我不后悔，70块换个头像，值了！）。
  2. 成功的在开发机上用vim开发，再也不用IDE吃我电脑为数不多的内存了。
  3. 在考虑买下Alfred，感觉非常丝滑。
  4. mathpix snipping tool 真的好用，再也不用手打公式了，甚至可以当简化版的OCR用。
  5. sublime + skim 写latex非常爽，非常丝滑
  6. magnet不是非常好用，感觉买亏了
  7. XCode如果实在太闲的话，可以转一下，iOS开发不难的。
  8. Karabiner-Elements 一定要改键位，改完之后直接起飞
    1. caps_lock -> left_control
    2. Left ctrl+hjkl to arrow keys Vim
    3. Post escape if left_control is pressed alone.
      …
      n. 是的，上面都是在记录我用过的摸鱼软件，哈哈哈哈哈

写完这段的时候，其实有点悲凉的意味了，如果我想记录的美好只剩下一些办公软件的话，说明做业务确实没有什么值得开心的事。
