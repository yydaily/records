B. MEXor Mixup

开启传送门

题目描述

Alice gave Bob two integers aaa and bbb (a>0a > 0a>0 and b≥0b \ge 0b≥0). Being a curious boy, Bob wrote down an array of non-negative integers with MEX⁡\operatorname{MEX}MEX value of all elements equal to aaa and XOR⁡\operatorname{XOR}XOR value of all elements equal to bbb.

What is the shortest possible length of the array Bob wrote?

Recall that the MEX⁡\operatorname{MEX}MEX () of an array is the minimum non-negative integer that does not belong to the array and the XOR⁡\operatorname{XOR}XOR of an array is the bitwise XOR of all the elements of the array.

Input

The input consists of multiple test cases. The first line contains an integer ttt (1≤t≤5⋅1041 \leq t \leq 5 \cdot 10^41≤t≤5⋅104) — the number of test cases. The description of the test cases follows.

The only line of each test case contains two integers aaa and bbb (1≤a≤3⋅1051 \leq a \leq 3 \cdot 10^51≤a≤3⋅105; 0≤b≤3⋅1050 \leq b \leq 3 \cdot 10^50≤b≤3⋅105) — the MEX⁡\operatorname{MEX}MEX and XOR⁡\operatorname{XOR}XOR of the array, respectively.

Output

For each test case, output one (positive) integer — the length of the shortest array with MEX⁡\operatorname{MEX}MEX aaa and XOR⁡\operatorname{XOR}XOR bbb. We can show that such an array always exists.

Example

Input1

5
1 1
2 1
2 0
1 10000
2 10000

Output1

3
2
3
2
3

Note

In the first test case, one of the shortest arrays with MEX⁡\operatorname{MEX}MEX 111 and XOR⁡\operatorname{XOR}XOR 111 is [0,2020,2021][0, 2020, 2021][0,2020,2021].

In the second test case, one of the shortest arrays with MEX⁡\operatorname{MEX}MEX 222 and XOR⁡\operatorname{XOR}XOR 111 is [0,1][0, 1][0,1].

It can be shown that these arrays are the shortest arrays possible.

题意

多组样例，每次给你两个数字 aaa 和 bbb。

让你找到一个长度最短的非负整数数列c1,c2,…,cnc_1, c_2, \dots, c_nc1​,c2​,…,cn​。满足

  1. 最小的没出现在数列中的非负整数为 aaa.
  2. c1⊕c2⊕⋯⊕cn=bc_1 \oplus c_2 \oplus \dots \oplus c_n = bc1​⊕c2​⊕⋯⊕cn​=b

分析

首先为了满足条件111，我们显然一定需要使用 aaa 个数字 [0,1,…,a−1][0, 1, \dots, a-1][0,1,…,a−1]。

然后我们定义 x=0⊕1⊕2⊕⋯⊕(a−1)x = 0\oplus 1\oplus 2\oplus \dots \oplus (a-1)x=0⊕1⊕2⊕⋯⊕(a−1)。

有以下三条结论：

  1. 如果 x==bx==bx==b 那么显然有答案为aaa
  2. 如果 x⊕b!=ax\oplus b != ax⊕b!=a 那么答案为 a+1a+1a+1
  3. 否则答案为 a+2a+2a+2。

下面是证明：

如果 x==bx==bx==b 那么显然有答案为aaa

  • 显然，我只需要选上述的序列即可。

如果 x⊕b!=ax\oplus b != ax⊕b!=a 那么答案为 a+1a+1a+1

  • 我只需要选择 [0,1,…,a−1,0⊕1⊕2⊕⋯⊕(a−1)⊕b][0, 1, \dots, a-1, 0\oplus 1\oplus 2\oplus \dots \oplus (a-1)\oplus b][0,1,…,a−1,0⊕1⊕2⊕⋯⊕(a−1)⊕b]，那么有 x=bx=bx=b，并且由于 0⊕1⊕⋯⊕(a−1)⊕b!=a0\oplus 1\oplus \dots\oplus (a-1)\oplus b != a0⊕1⊕⋯⊕(a−1)⊕b!=a，所以 aaa就是没出现过的最小的非负整数，所以这是一个最小解。

否则答案为 a+2a+2a+2。

  • 先证明存在性
    • 我只需要选择 [0,1,…,a−1,0⊕1⊕⋯⊕(a−1)⊕b+1<<100,1<<100][0, 1, \dots, a-1, 0\oplus 1\oplus \dots\oplus (a-1)\oplus b + 1<<100, 1<<100][0,1,…,a−1,0⊕1⊕⋯⊕(a−1)⊕b+1<<100,1<<100], 显然有 x=bx = bx=b。
  • 再证明最小
    • 前面我们有了aaa 个数是必选的，所以我只需要证明不存在长度为 a+1a+1a+1 的解即可
    • 采用反证法，我们假设多的这个数字为 yyy，因为 aaa 不存在，所以 y!=ay!=ay!=a。
    • 我们选择了 [0,1,…,a−1,y][0, 1, \dots, a-1, y][0,1,…,a−1,y]，因为 0⊕1⊕⋯⊕(a−1)=a⊕b0\oplus 1\oplus \dots \oplus (a-1) = a\oplus b0⊕1⊕⋯⊕(a−1)=a⊕b，并且有 0⊕1⊕⋯⊕(a−1)⊕y=b0\oplus 1\oplus\dots\oplus (a-1)\oplus y = b0⊕1⊕⋯⊕(a−1)⊕y=b，这两个式子异或可以得到 y=ay = ay=a，这显然和 y!=ay!=ay!=a相悖。所以不存在长度为 a−1a-1a−1 的解，原命题得证。
