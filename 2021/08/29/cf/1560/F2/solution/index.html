F2. Nearest Beautiful Number (hard version)

开启传送门

题目描述

It is a complicated version of problem F1. The difference between them is the constraints (F1: k≤2k \le 2k≤2, F2: k≤10k \le 10k≤10).

You are given an integer nnn. Find the minimum integer xxx such that x≥nx \ge nx≥n and the number xxx is kkk-beautiful.

A number is called kkk-beautiful if its decimal representation having no leading zeroes contains no more than kkk different digits. E.g. if k=2k = 2k=2, the numbers 343444334344433434443, 555505555055550, 777777777 and 212121 are kkk-beautiful whereas the numbers 120120120, 445435445435445435 and 998244353998244353998244353 are not.

Input

The first line contains one integer ttt (1≤t≤1041 \le t \le 10^41≤t≤104) — the number of test cases. Then ttt test cases follow.

Each test case consists of one line containing two integers nnn and kkk (1≤n≤1091 \le n \le 10^91≤n≤109, 1≤k≤101 \le k \le 101≤k≤10).

Output

For each test case output on a separate line xxx — the minimum kkk-beautiful integer such that x≥nx \ge nx≥n.

Example

Input1

6
2021 3
177890 2
34512 3
724533 4
998244353 1
12345678 10

Output1

2021
181111
34533
724542
999999999
12345678

题意

多组用例，每组给你两个整数 nnn 和 kkk，你需要找到一个整数 xxx ，满足 n≤xn\le xn≤x 且 xxx 在十进制表示下，按位拆分，有不超过 kkk 个不同的数字。

分析

裸dfs，每次枚举一个当前最小的数字，然后dfs下去，看是否满足，如果不满足就 +1+1+1 。
