E. Mocha and Stars

开启传送门

题目描述

Mocha wants to be an astrologer. There are nnn stars which can be seen in Zhijiang, and the brightness of the iii-th star is aia_iai​.

Mocha considers that these nnn stars form a constellation, and she uses (a1,a2,…,an)(a_1,a_2,\ldots,a_n)(a1​,a2​,…,an​) to show its state. A state is called mathematical if all of the following three conditions are satisfied:

  • For all iii (1≤i≤n1\le i\le n1≤i≤n), aia_iai​ is an integer in the range [li,ri][l_i, r_i][li​,ri​].

  • ∑i=1nai≤m\sum \limits _{i=1} ^ n a_i \le mi=1∑n​ai​≤m.

  • gcd⁡(a1,a2,…,an)=1\gcd(a_1,a_2,\ldots,a_n)=1gcd(a1​,a2​,…,an​)=1.

Here, gcd⁡(a1,a2,…,an)\gcd(a_1,a_2,\ldots,a_n)gcd(a1​,a2​,…,an​) denotes the greatest common divisor (GCD) of integers a1,a2,…,ana_1,a_2,\ldots,a_na1​,a2​,…,an​.

Mocha is wondering how many different mathematical states of this constellation exist. Because the answer may be large, you must find it modulo 998 244 353998\,244\,353998244353.

Two states (a1,a2,…,an)(a_1,a_2,\ldots,a_n)(a1​,a2​,…,an​) and (b1,b2,…,bn)(b_1,b_2,\ldots,b_n)(b1​,b2​,…,bn​) are considered different if there exists iii (1≤i≤n1\le i\le n1≤i≤n) such that ai≠bia_i \ne b_iai​​=bi​.

Input

The first line contains two integers nnn and mmm (2≤n≤502 \le n \le 502≤n≤50, 1≤m≤1051 \le m \le 10^51≤m≤105) — the number of stars and the upper bound of the sum of the brightness of stars.

Each of the next nnn lines contains two integers lil_ili​ and rir_iri​ (1≤li≤ri≤m1 \le l_i \le r_i \le m1≤li​≤ri​≤m) — the range of the brightness of the iii-th star.

Output

Print a single integer — the number of different mathematical states of this constellation, modulo 998 244 353998\,244\,353998244353.

Note

  • a1=1,a2=1.a_1 = 1, a_2 = 1.a1​=1,a2​=1.
  • a1=1,a2=2.a_1 = 1, a_2 = 2.a1​=1,a2​=2.
  • a1=2,a2=1.a_1 = 2, a_2 = 1.a1​=2,a2​=1.
  • a1=3,a2=1.a_1 = 3, a_2 = 1.a1​=3,a2​=1.

题意

给你一个长度为 nnn 的二元组 [li,ri][l_i, r_i][li​,ri​]，和一个上限 mmm.

让你找到有多少个长度为 nnn 的序列 [a1,a2…,an][a_1, a_2\dots, a_n][a1​,a2​…,an​]，满足一下条件：

  • ∀i,ai∈[li,ri]\forall i , a_i \in [l_i, r_i]∀i,ai​∈[li​,ri​]
  • ∑i=1nai≤m\sum_{i=1}^n a_i \le m∑i=1n​ai​≤m
  • gcd(a1,a2,…,an)=1gcd(a_1, a_2, \dots, a_n) = 1gcd(a1​,a2​,…,an​)=1

分析

如果不考虑最后一个条件，这个题应该怎么做呢？

我们考虑定义一个函数 f(l1,r1,…,ln,rn,m)f(l_1, r_1, \dots, l_n, r_n, m)f(l1​,r1​,…,ln​,rn​,m)，接手 2n+12n+12n+1 个参数，分别为 nnn 个区间，和上限。输出满足前两个条件的序列数。那么我们可以得到以下递推公式

f(l1,r1,…,ln,rn,m)=∑i=lnrnf(l1,r1,…,ln−1,rn−1,m−i)f(l_1, r_1, \dots, l_n, r_n, m) = \sum_{i=l_n}^{r_n} f(l_1, r_1, \dots, l_{n-1}, r_{n-1}, m-i)f(l1​,r1​,…,ln​,rn​,m)=∑i=ln​rn​​f(l1​,r1​,…,ln−1​,rn−1​,m−i)

所以我们可以定义函数 dp[i][j]dp[i][j]dp[i][j] 表示前 iii 个数字，求和为 jjj 的方案数。sum[i][j]sum[i][j]sum[i][j] 表示前 iii 个数字，求和为 ≤j\le j≤j 的方案数。然后就可以 O(n∗m)O(n*m)O(n∗m) 的递推这个数组，得到答案。

然后还有一个空间优化就是，我们可以很显然的发现长度为 nnn 的答案，一定是由长度为 n−1n-1n−1 递推过来的，所以我们可以省掉一个维度，所以上述问题可以 O(m)O(m)O(m) 的空间，O(n∗m)O(n*m)O(n∗m) 的时间得到答案。

最后，加上条件 333 我们怎么统计答案呢？我们可以得到答案为下述式子：

\begin{align*} &f(l_1, r_1, \dots, l_n, r_n, m) * [gcd(a_1, \dots, a_n) == 1]\\ = &\sum_{a_1 = l_1}^{r_1} \sum_{a_2 = l_2}^{r_2}\dots\sum_{a_n = l_n}^{r_n} [\sum_{i=1}^n a_i \le m] * [gcd(a_1, \dots, a_n) == 1]\\ = &\sum_{a_1 = l_1}^{r_1} \sum_{a_2 = l_2}^{r_2}\dots\sum_{a_n = l_n}^{r_n} [\sum_{i=1}^n a_i \le m] * \sum_{d|a_1, d|a_2, \dots, d|a_n} \mu(d)\\ = & \sum_{d=1}^{m} \mu(d) \sum_{a_1=\lceil \frac{l_1}{d}\rceil}^{\lfloor\frac{r_1}{d}\rfloor} \sum_{a_2=\lceil\frac{l_2}{d}\rceil}^{\lfloor\frac{r_2}{d}\rfloor}\dots \sum_{a_n=\lceil\frac{l_n}{d}\rceil}^{\lfloor\frac{r_n}{d}\rfloor} [\sum_{i=1}^n a_i\le \lfloor \frac{m}{d}\rfloor] \end{align*}

根据上述式子，我们能得到后一部分 也即

∑a1=⌈l1d⌉⌊r1d⌋∑a2=⌈l2d⌉⌊r2d⌋⋯∑an=⌈lnd⌉⌊rnd⌋[∑i=1nai≤⌊md⌋]=f(⌈l1d⌉,⌊r1d⌋,…,⌈lnd⌉,⌊rnd⌋,⌊md⌋)\sum_{a_1=\lceil \frac{l_1}{d}\rceil}^{\lfloor\frac{r_1}{d}\rfloor} \sum_{a_2=\lceil\frac{l_2}{d}\rceil}^{\lfloor\frac{r_2}{d}\rfloor}\dots \sum_{a_n=\lceil\frac{l_n}{d}\rceil}^{\lfloor\frac{r_n}{d}\rfloor} [\sum_{i=1}^n a_i\le \lfloor \frac{m}{d}\rfloor] = f(\lceil \frac{l_1}{d}\rceil, \lfloor\frac{r_1}{d}\rfloor, \dots, \lceil\frac{l_n}{d}\rceil, \lfloor\frac{r_n}{d}\rfloor, \lfloor \frac{m}{d}\rfloor) a1​=⌈dl1​​⌉∑⌊dr1​​⌋​a2​=⌈dl2​​⌉∑⌊dr2​​⌋​⋯an​=⌈dln​​⌉∑⌊drn​​⌋​[i=1∑n​ai​≤⌊dm​⌋]=f(⌈dl1​​⌉,⌊dr1​​⌋,…,⌈dln​​⌉,⌊drn​​⌋,⌊dm​⌋)
