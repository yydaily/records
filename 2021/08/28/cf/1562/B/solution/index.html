B. Scenes From a Memory

开启传送门

题目描述

During the hypnosis session, Nicholas suddenly remembered a positive integer nnn, which doesn’t contain zeros in decimal notation.

Soon, when he returned home, he got curious: what is the maximum number of digits that can be removed from the number so that the number becomes not prime, that is, either composite or equal to one?

For some numbers doing so is impossible: for example, for number 535353 it’s impossible to delete some of its digits to obtain a not prime integer. However, for all nnn in the test cases of this problem, it’s guaranteed that it’s possible to delete some of their digits to obtain a not prime number.

Note that you cannot remove all the digits from the number.

A prime number is a number that has no divisors except one and itself. A composite is a number that has more than two divisors. 111 is neither a prime nor a composite number.

Input

Each test contains multiple test cases.

The first line contains one positive integer ttt (1≤t≤1031 \le t \le 10^31≤t≤103), denoting the number of test cases. Description of the test cases follows.

The first line of each test case contains one positive integer kkk (1≤k≤501 \le k \le 501≤k≤50) — the number of digits in the number.

The second line of each test case contains a positive integer nnn, which doesn’t contain zeros in decimal notation (10k−1≤n<10k10^{k-1} \le n < 10^{k}10k−1≤n<10k). It is guaranteed that it is always possible to remove less than kkk digits to make the number not prime.

It is guaranteed that the sum of kkk over all test cases does not exceed 10410^4104.

Output

For every test case, print two numbers in two lines. In the first line print the number of digits, that you have left in the number. In the second line print the digits left after all delitions.

If there are multiple solutions, print any.

Note

In the first test case, you can’t delete 222 digits from the number 237237237, as all the numbers 222, 333, and 777 are prime. However, you can delete 111 digit, obtaining a number 27=3327 = 3^327=33.

In the second test case, you can delete all digits except one, as 4=224 = 2^24=22 is a composite number.

题意

ttt 次询问。每次给你一个长度为 kkk 的字符串，问你能不能删掉几个字符后，剩下的字符组成的数字是个合数。要求删掉的字符尽可能的多。保证原始串不含 000。

分析

我们能容易的得到答案不超过三位。下面是证明：

首先我们有，如果一个数字是 333 的倍数，那么这个数的各位和也一定是 333 的倍数。

所以我们不妨把数字按照 %3\%3%3 的余数分组。也即 [1,4,7],[2,5,8],[3,6,9][1,4,7], [2,5,8], [3,6,9][1,4,7],[2,5,8],[3,6,9]。

其中第 111 组中的 444，第 222 组中的 888 和第 333 组中的 6,96,96,9 本身就是合数，所以这种case可以直接输出答案，先排除。剩下的数字就是：[1,7],[2,5],[3][1, 7], [2, 5], [3][1,7],[2,5],[3]

任何以 2,52, 52,5 结尾的 两位数都是合数，所以直接排除。剩下的数字就是 [1,7],[3][1, 7], [3][1,7],[3]。

接下来用反证法，我们假设 存在答案长度至少为4。因为 333333 是合数，所以 最后一组 只能出现一次，也即 [1,7][1,7][1,7] 至少出现了 333 次，而根据最前面的假设 111 和 777 出现了三次，仅这三个数字就是 333 的倍数了。 所以不存在答案长度超过 333 的。

剩下的就简单了，直接枚举长度为 111 222 和 333 的所有数字，判断是否为质数即可。
