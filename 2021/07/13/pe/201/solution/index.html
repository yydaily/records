For any set A of numbers, let sum(A) be the sum of the elements of A.
Consider the set B = {1,3,6,8,10,11}.
There are 20 subsets of B containing three elements, and their sums are:

sum({1,3,6}) = 10,
sum({1,3,8}) = 12,
sum({1,3,10}) = 14,
sum({1,3,11}) = 15,
sum({1,6,8}) = 15,
sum({1,6,10}) = 17,
sum({1,6,11}) = 18,
sum({1,8,10}) = 19,
sum({1,8,11}) = 20,
sum({1,10,11}) = 22,
sum({3,6,8}) = 17,
sum({3,6,10}) = 19,
sum({3,6,11}) = 20,
sum({3,8,10}) = 21,
sum({3,8,11}) = 22,
sum({3,10,11}) = 24,
sum({6,8,10}) = 24,
sum({6,8,11}) = 25,
sum({6,10,11}) = 27,
sum({8,10,11}) = 29.

Some of these sums occur more than once, others are unique.
For a set A, let U(A,k) be the set of unique sums of k-element subsets of A, in our example we find U(B,3) = {10,12,14,18,21,25,27,29} and sum(U(B,3)) = 156.

Now consider the 100-element set S = {12,22,...,10021^2, 2^2, ... , 100^212,22,...,1002}.
S has 100891344545564193334812497256 50-element subsets.

Determine the sum of all integers which are the sum of exactly one of the 50-element subsets of S, i.e. find sum(U(S,50)).

translation

对于任何整数集合 A，定义 sum(A) 为集合 A 中每个元素的和。
例如集合 B = {1,3,6,8,10,11}。
其中 B 有 20 个包含三个元素的自己，并且他们的和如下：

sum({1,3,6}) = 10,
sum({1,3,8}) = 12,
sum({1,3,10}) = 14,
sum({1,3,11}) = 15,
sum({1,6,8}) = 15,
sum({1,6,10}) = 17,
sum({1,6,11}) = 18,
sum({1,8,10}) = 19,
sum({1,8,11}) = 20,
sum({1,10,11}) = 22,
sum({3,6,8}) = 17,
sum({3,6,10}) = 19,
sum({3,6,11}) = 20,
sum({3,8,10}) = 21,
sum({3,8,11}) = 22,
sum({3,10,11}) = 24,
sum({6,8,10}) = 24,
sum({6,8,11}) = 25,
sum({6,10,11}) = 27,
sum({8,10,11}) = 29.

其中一些和出现了不止一次，其他的则仅出现一次。
对于一个集合 A，定义 U(A, k) 表示 A 的 k 个元素的子集中，子集和只出现一次的子集个数，例如上面的例子，我们能发现 U(B, 3) = {10,12,14,18,21,25,27,19}，并且有 sum(U(B,3)) = 156。

现在考虑一个有100个元素的集合 S = {12,22,...,10021^2, 2^2, ... , 100^212,22,...,1002}。
S 有 100891344545564193334812497256 个包含50个元素的子集。

这些子集的元素和，其中一些和只出现了一次，求这些出现一次的和的和。也即，求 sum(U(S, 50))。
