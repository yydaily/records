线性同余方程

2021-10-15
Word count: 884   |   Reading time≈ 4 min

此篇文章收录于算法笔记中，建议提前阅读前缀知识。

定义

在数论中，线性同余方程是最基本的同余方程，“线性”表示方程的未知数次数是一次，即形如：ax≡c  ( mod b)ax \equiv c\ \ (\bmod b)ax≡c  (modb) 的方程。

解法

ax≡c  ( mod b)ax \equiv c\ \ (\bmod b)ax≡c  (modb) 可以等价为 ax+by≡c  ( mod b)ax + by \equiv c\ \ (\bmod b)ax+by≡c  (modb) .

所以原问题转化为解这个方程。下面有两个推论。

推论0：ax+byax + byax+by 在  mod b\bmod bmodb 意义下，能组成的最小正整数是 gcd(a,b)gcd(a,b)gcd(a,b)

证明

原命题的意思是，∀x,min{ax mod b>0∣ax mod b}=gcd(a,b)\forall x, min\{ax\bmod b > 0 | ax\bmod b\} = gcd(a, b)∀x,min{axmodb>0∣axmodb}=gcd(a,b)

根据扩展欧几里得定理，一定存在 x0x_0x0​ 使得 ax0≡gcd(a,b) ( mod b)ax_0 \equiv gcd(a, b)\ (\bmod b)ax0​≡gcd(a,b) (modb)

采用反证法，我们假设存在 x1x_1x1​ 使得 ax1≡t ( mod b)ax_1 \equiv t\ (\bmod b)ax1​≡t (modb) 并且有 $t \gt 0 \land t \lt gcd(a, b) $ ，也即最小整数不是 gcd(a,b)gcd(a,b)gcd(a,b)

显然我们有 gcd(a,b)∣a∣ax1gcd(a,b)\mid a\mid ax_1gcd(a,b)∣a∣ax1​ 并且有 gcd(a,b)∣bgcd(a, b) \mid bgcd(a,b)∣b ，所以我们有 gcd(a,b)∣tgcd(a, b) \mid tgcd(a,b)∣t ，也即 $t = x * gcd(a, b) \gt gcd(a, b) $ 。与 t<gcd(a,b)t \lt gcd(a,b)t<gcd(a,b) 相悖，原命题得证。

推论1：方程有解的前提是 gcd(a,b)∣cgcd(a, b) | cgcd(a,b)∣c

证明

还是反证法，我们假设 gcd(a,b)∤cgcd(a,b) \nmid cgcd(a,b)∤c 。

我们定义 $d = c \bmod gcd(a,b) $ 。那么显然 d<gcd(a,b)d \lt gcd(a,b)d<gcd(a,b) 。

也即，我们需要找到一对 x1,y1x_1, y_1x1​,y1​ 使得 ax1+by1≡d ( mod b)ax_1 + by_1 \equiv d\ (\bmod b)ax1​+by1​≡d (modb) 。

根据推论 000 ，我们知道，这是可能的。原命题得证。

构造解

根据扩展欧几里得算法，我们可以找到一对 x0,y0x_0, y_0x0​,y0​ 使得 ax0+by0≡gcd(a,b) ( mod b)ax_0 + by_0 \equiv gcd(a,b)\ (\bmod b)ax0​+by0​≡gcd(a,b) (modb) 。

又因为 gcd(a,b)∣cgcd(a,b) \mid cgcd(a,b)∣c ，所以我们可以构造一组基本解 $ x_0 * c / gcd(a,b), y_0 * c / gcd(a,b) $

通解

我们现在有了一个解，如何扩展到所有的解呢？

一个直观的想法是，如果我们有 ax0+by0≡c ( mod b)ax_0 + by_0 \equiv c\ (\bmod b)ax0​+by0​≡c (modb) 。

我们可以令 x1=x0+bgcd(a,b)x_1 = x_0 + \frac{b}{gcd(a,b)}x1​=x0​+gcd(a,b)b​ , y1=y0−agcd(a,b)y_1 = y_0 - \frac{a}{gcd(a,b)}y1​=y0​−gcd(a,b)a​ 。

更一般的，我们令 xk=x0+b∗kgcd(a,b),yk=y0−a∗kgcd(a,b),k∈Zx_k = x_0 + \frac{b*k}{gcd(a,b)}, y_k = y_0 - \frac{a*k}{gcd(a,b)}, k\in \mathbb{Z}xk​=x0​+gcd(a,b)b∗k​,yk​=y0​−gcd(a,b)a∗k​,k∈Z 便能得到所有的解。证明如下：

axk+byk=ax0+a∗b∗kgcd(a,b)+by0−a∗b∗kgcd(a,b)=ax0+by0≡c ( mod b)ax_k + by_k = ax_0 + \frac{a*b*k}{gcd(a,b)} + by_0 - \frac{a*b*k}{gcd(a,b)} = ax_0 + by_0 \equiv c\ (\bmod b)axk​+byk​=ax0​+gcd(a,b)a∗b∗k​+by0​−gcd(a,b)a∗b∗k​=ax0​+by0​≡c (modb)

命题得证.

最小正整数解

因为 xk=x0+b∗kgcd(a,b),k∈Zx_k = x_0 + \frac{b*k}{gcd(a,b)}, k \in \mathbb{Z}xk​=x0​+gcd(a,b)b∗k​,k∈Z 。所以，只需要计算 xmin=x0%bgcd(a,b)x_{min} = x_0 \% \frac{b}{gcd(a,b)}xmin​=x0​%gcd(a,b)b​ 即可得到最小解。
