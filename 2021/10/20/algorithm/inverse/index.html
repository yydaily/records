定义

逆元也称为 模逆元 ，一个整数 aaa 和同余 nnn 的逆元是指满足一下公式的整数 bbb :

a−1≡b   ( mod n)a^{-1} \equiv b \ \ \ (\bmod n) a−1≡b   (modn)

也可以写成：

ab≡1   ( mod n)ab \equiv 1\ \ \ (\bmod n) ab≡1   (modn)

逆元存在的充要条件是 aaa 和 nnn 互素。

证明

充要性

也即证明 ∃b,ab≡1   ( mod n)⇒gcd(a,n)=1\exists b, ab\equiv 1\ \ \ (\bmod n) \Rightarrow gcd(a,n)=1∃b,ab≡1   (modn)⇒gcd(a,n)=1

\begin{align} &\exists b, ab\equiv 1 (\bmod n)\\ \Rightarrow& \exists x,\exists y, ax + ny \equiv 1 (\bmod n)\\ \Rightarrow& gcd(a, n) \le 1\\ \Rightarrow& gcd(a, n) = 1 \end{align}

其中有一步的原因是，我们在扩展欧几里得里面提到过 ax+nyax+nyax+ny 能组成的最小正整数为 gcd(a,n)gcd(a, n)gcd(a,n) ，所以如果能组成 111 ，那么说明 gcd(a,n)≤1gcd(a, n) \le 1gcd(a,n)≤1 。

必要性

也即证明 gcd(a,n)=1⇒∃b,ab≡1( mod n)gcd(a, n) = 1 \Rightarrow \exists b, ab \equiv 1 (\bmod n)gcd(a,n)=1⇒∃b,ab≡1(modn) 。

根据扩展欧几里得，一定存在 (x0,y0)(x_0, y_0)(x0​,y0​) 使得 ax0+ny0≡1( mod n)ax_0 + ny_0 \equiv 1 (\bmod n)ax0​+ny0​≡1(modn) 。这里我们令 b=x0b=x_0b=x0​ 即可。

命题得证。

为什么需要逆元

其实在我学会逆元的很长时间里面，我都没有意识到这个问题，以至于对初学者来说，这个可能是一个纯概念性的东西，而无法正确理解。

如果不想知道的同学，可以直接略过这块，毕竟是个人见解，不太重要。

其实原因很简单，下面我们看几个式子（假设 a,b,pa,b,pa,b,p 都是正整数，负数同样可证，这里略过）：

先说结论，前三个都是对的，最后一个是错的。下面是证明

[O] (a+b)%p=((a%p)+(b%p))%p(a+b)\%p = ((a\%p) + (b\%p))\%p(a+b)%p=((a%p)+(b%p))%p

单个计算

扩展欧几里得算法

aaa 对 %n\%n%n 逆元存在的充要条件是，gcd(a,n)=1gcd(a, n)=1gcd(a,n)=1 ，所以一个自然的想法是用扩展欧几里得来计算逆元，因为本质上，扩展欧几里得是在解不定方程：

ax+ny≡gcd(a,n)( mod n)ax+ny\equiv gcd(a,n) (\bmod n) ax+ny≡gcd(a,n)(modn)

所以我们只需要知道最小的 x0x_0x0​ ，即是 aaa 的逆元。

欧拉定理

和欧几里得类似，因为有 gcd(a,n)=1gcd(a,n)=1gcd(a,n)=1 ，所以我们可以直接套用欧几里得定理：

aφ(n)≡1( mod n)a^{\varphi(n)} \equiv 1 (\bmod n) aφ(n)≡1(modn)

费马小定理

更进一步的，比赛的时候，经常会给出 nnn 是个质数，对于这种情况，我们通常会用费马小定理+快速幂，快速的算出一个数的逆元。也即

a−1=an−1( mod n)a^{-1} = a^{n-1} (\bmod n) a−1=an−1(modn)

批量计算

不难看出，上述的方法虽然适用于单个逆元的计算，但是复杂度是 O(logn)O(log n)O(logn) 的，也即如果我要计算 对于 ∀a∈[1,n],a−1%n\forall a \in [1, n], a^{-1} \% n∀a∈[1,n],a−1%n ，整体复杂度就到了 nlog(n)nlog(n)nlog(n) ，有没有更快的做法呢？

当然有，不然我这BB一顿，浪费我的时间也浪费读者时间？

给出递推式（没有明确指出的情况下，所有的除法都是不带小数点的整除）：

inv[i]=(n−n/i)∗inv[n%i]%ninv[i] = (n-n/i) * inv[n\%i] \%n inv[i]=(n−n/i)∗inv[n%i]%n

这样，我们从 111 到 nnn 来枚举 iii ，直接递推就能得到所有的逆元。下面给出证明：

\begin{align} &n/i*i + n\%i \equiv 0 &(\bmod n) \\ \Rightarrow &n\%i \equiv -n/i*i &(\bmod n)\\ \Rightarrow &n\%i*inv[i] \equiv -n/i &(\bmod n)\\ \Rightarrow &inv[i] \equiv -n/i*inv[n\%i] &(\bmod n)\\ \Rightarrow &inv[i] \equiv (n-n/i)*inv[n\%i] &(\bmod n)\\ \end{align}

原命题得证。
